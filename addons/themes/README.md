# themes

[api] = 2
[version] = 1.0.0
[authors] = {"Aarav Maloo"}
[description] = <Select and apply built-in themes (default, dark, light, dracula, solarized)>
[keybind] = c
[mode] = workspace
[permissions] = {"write-notes"}

A built-in theme switcher that lists available color presets and applies the selected one to blob's config file. Requires a restart of blob to take effect.
